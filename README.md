# Security-Analytics-Project
Final Research Project on LLM Prompt Injection Analysis using Traditional ML Methods

The full paper for my final research project can be found [here](./Security_DA_Project.pdf)


After completing the full coursework on topics such as Clustering and Classification, Anomaly Detection, Text Analytics, Deep Learning, and Graph Analytics, I completed this final research project in which I use particular analytic technique to a data set in comparison to current state of the art in a particular cybersecurity activity.

This research project focuses on Large Language Models
(LLMs) and how the problem of prompt injection can be prevented through machine learning classification for malicious prompt injections. The detection of adversarial prompt injections using a simple classification model can demonstrate the robustness and reliability of detecting prompt injection attacks given limited computational power and data capacity. 

The jupyter notebook with content to complete my analysis and testing can be found [here](./LogisticRegressionClassifier_Eval.ipynb)
